made with https://quarto.org/

requires `quarto install extension sellorm/quarto-social-embeds`

to build _site use `quarto render`

social embeds support:
- `{{< gist USERNAME GIST_ID >}}`
- `{{< youtube VIDEO_ID >}}`
- `{{< vimeo VIDEO_ID >}}`
- `{{< loom VIDEO_ID >}}`
- `{{< tweet USERNAME TWEET_ID >}}`
- `{{< mastodon STATUS_URL >}}`
